# Animated React Card UI
A React + Tailwind CSS project focused on building animated UI cards with dynamic props, gradient animations, reusable components, and responsive layouts.

This project was created as a hands-on practice project to improve understanding of React fundamentals, props handling, array rendering, conditional logic, and advanced Tailwind styling.

# Features
- Dynamic card rendering using props
- Animated gradient borders
- Reusable React components
- Responsive grid layout
- Auto-generated missing cards
- Tailwind CSS styling
- Smooth animated backgrounds
- Data-driven UI rendering

# Technologies Used

- Tailwind CSS
- JavaScript (ES6)
